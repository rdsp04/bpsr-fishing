Welcome to BPSR Fishing

A quick start guide and reference for new users

Instructions

  1. Start Blue Protocol: Star Resonance.
  2. Open BPSR Fishing.
  3. Set the game window to 1920x1080 resolution (Fullscreen or windowed works).
  4. Ensure your character is in a fishing position before starting.
  5. Check your amount of baits and rods; if unsure, see Recommendations.
  6. Now press F9 and enjoy your fishing XP!

Usage

  • Press F9 to start the macro.
  • Press F10 to stop the macro.
  • The script tracks catches, fish types, XP, and sessions automatically.

FAQ

  • If the script presses the Exit button instead of Continue, restart it.
  • Launch the script after the game is opened.
  • The game must be placed on the main monitor.
  • The window must be visible for the script to work properly.
  • To open the script, type bpsr-fishing in Windows search.
